# Settings 

 > what are Addets?

Addets are the options that are available to the user when creating a challenge or tournament.

<br>

# Types of Addets 

 > Visibility
  * Public / open-for-all
  * Private / invite only

<br>

 > Member limit 
 * No limit
 * 1-10000...

 <br>

 > Time limit / duration
  * No limit
  * 1 months (30 days) to 1 year (365 days)

<br>

 > Game type
 * Tournament
 * Challenge
 * Mini-game
